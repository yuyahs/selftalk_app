# Overview

* どういうサイトか：アウトプット特化の英語学習用サイト
* 何ができるか: ①ランダムに問題文が表示され、英作文で回答する
②作成した回答の添削
③オリジナルの単語帳作成
* アピールポイント: Rails × Vue3によるSPA化実装

# 使用技術

* Ruby 3.0.2
* Ruby on Rails 6.0,4
* MySQL 5.7
* Vue.js 3
* Nginx
* Unicorn
* AWS:
    VPC
    EC2
    Route53
* Capistrano3
* RSpec

# 機能一覧
* ユーザー登録（メール認証）、ログイン機能
* ゲストログイン機能
* お知らせ通知機能
* 問題出題機能、回答機能
* 回答添削機能
* 単語帳作成機能
